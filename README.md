#### Desafio Sensio


**Iniciando o projeto**

* **Docker**
  <p>Iniciar o banco de dados</p>
```
docker-compose up -d
```

* **API**
```
start => yarn install && yarn dev
```

* **Web**
  
```
start => yarn install && yarn start
```

* **Observações**
  
  Verificar se existe o arquivo **.env.example**, caso sim, modificar para **.env** e preencher as informações necessárias.
